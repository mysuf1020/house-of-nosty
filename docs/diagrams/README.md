# Diagram Sistem - House of Nosty E-Menu

Folder ini berisi diagram-diagram untuk dokumentasi skripsi.

## Cara Generate Gambar

1. Buka https://mermaid.live/
2. Copy-paste kode dari file `.mmd` 
3. Download sebagai PNG/SVG

## Daftar Diagram

1. **ERD (Entity Relationship Diagram)** - `erd.mmd`
2. **Use Case Diagram** - `use-case.mmd`
3. **Activity Diagram - Pemesanan** - `activity-order.mmd`
4. **Activity Diagram - Admin** - `activity-admin.mmd`
5. **Sequence Diagram - Order Flow** - `sequence-order.mmd`
6. **Class Diagram** - `class-diagram.mmd`
7. **Flowchart Sistem** - `flowchart.mmd`
8. **Architecture Diagram** - `architecture.mmd`
